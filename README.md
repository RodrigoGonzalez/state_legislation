#A Search Engine for State Legislation

##Overview:
This is an ongoing project to collect the text of all bills voted on by state legislatures and make them accessible for easy searching. Metadata about the bills comes from Sunlight Foundation's OpenStates Project, and the texts of the bills are from each state legislature's website.
